# Complete Project Overview

This is a portfolio website for a design agency/individual named **Ariths**.

## 🎯 Project Purpose

A portfolio website showcasing work, services, and contact information for a design professional/agency specializing in branding, web design, and Webflow development.

## 📁 Project Structure

* **Home Page (index.html)**
* **Contact Page (contact.html)**

## 🔑 Key Features

### 1. Home Page (index.html)

* Hero section with animated text
* Pitch slider showcasing brand values (Tight, Clean, Unique, Bold, Unexpected, Ethical, Growing, Luxury, Fierce, Hawt)
* Selected work portfolio featuring:

  * ArithHive
  * ArithVibe
  * TrueDay
* Services section:

  * Branding
  * Web Design
  * Webflow Development
* Client logos grid
* About section
* Call-to-Action (CTA) section
* Footer with social media links
* Custom cursor
* GSAP-powered scroll animations
* Smooth page transitions

### 2. Contact Page (contact.html)

* Contact form with the following fields:

  * Name
  * Email
  * Phone
  * Organization Name
  * Current Website
  * Service Selection (Branding / Web Design / Webflow)
* Alternative email contact option
* Consistent navigation and custom cursor

## 🎨 Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (including Webflow base styles)
* **GSAP (GreenSock Animation Platform)** – Scroll animations and transitions
* **Lenis** – Smooth scrolling
* **Webflow** – Website foundation
* **Google reCAPTCHA** – Form security
* **Google Analytics** – Website analytics
* **Finsweet Cookie Consent** – GDPR compliance

## 🖼️ Assets

The project includes:

* Logos
* Project screenshots
* Social media icons
* Decorative graphics
* Pattern images
* Embedded font files

## 📱 Responsive Design

Optimized for:

* Desktop (1920px+)
* Tablet (991px and below)
* Mobile (767px and below)
* Small Mobile (479px and below)

## 🎯 Brand Style

* **Colors:** CSS variables for consistent theming
* **Fonts:** Retail (sans-serif) and Swear (display/italic)
* **Layout:** Grid-based layout with smooth animations and transitions

This is a modern and responsive portfolio website designed to showcase creative work, highlight services, and generate client inquiries through an engaging user experience.
